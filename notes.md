# Notes

## General Purpose Registers 
these are the general purpose registers
- %eax
- %ebx
- %ecx
- %edx
- %edi
- %esx

these are special purpose registers 
- %ebp
- %esp
- %eip
- %eflags

## Instructions 
~~~
movl $1, %eax
~~~

moves value into a register 

~~~
int $0x80
~~~

interrupt command that invokes system call
